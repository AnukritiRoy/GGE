ASSIGNMENT 1

The scope of this Assignment is to get the students acquainted with the basic HTML encoding. Complete the following two tasks:

  1. Using a text editor, compile an HTML page (e.g., your personal page or a project home page). Include in this page: hyperlinks, images, lists, tables; and change the default formatting parameters using appropriate styles.
  2. Include in the same page an image or map with clickable areas and hyperlinks.

Upload the html and all accompanying documents (images, etc.) on your space at the course web server: http://gaia.gge.unb.ca/wmaps/students/ 

Send an email notification to your instructor (that you have completed the Assignment); upload a file (any file) on "D2L > Assessments > Assignment 1" to get access to Session 2 of the course.
